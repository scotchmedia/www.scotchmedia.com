### Deploying to heroku

1. Add the node version to `package.json`

  ```
  "engines": {
    "node": "0.8.x",
    "npm": "1.1.x"
  }

  ```

2. Create a `Procfile` with the command to start the server

  ```
  web: node app.js
  ```

3. Create a new heroku app:

  ```
  heroku create
  ```

4. Push the app to heroku:

  ```
  git push heroku master 
  ```

5. Test your creation:

  ```
  heroku open
  ```
